# Dino-Strike
🦕 Browser-based 3v3 tactical dino battle game with synergy system, status effects, and team draft mechanics

DinoStrike is a browser-based tactical battle game — no download, no install, just a link. Build a team of 3 dinos, equip them with gear, and fight through increasingly tough enemies using a real-time combat system with status effects, team synergies, and strategic pick/ban potential.

Features
3v3 team battle with synergy mechanics
Status effects — poison, burn, stun, shield
Visual gear loadout with rarity tiers
Survival mode + team mode
Runs on any browser, any device, any connection
Built with
Pure HTML, CSS, JavaScript — zero dependencies, zero frameworks

Roadmap
Multiplayer server, ranked mode, custom forum community, tournament brackets, and esports scene. Built by a competitive TCG player who understands meta.
